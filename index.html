AI + Urban Design Midterm

Reimagining Harlem Plaza as a greener, safer, and more welcoming public space.

This project studies the current plaza, identifies key design problems, and proposes an AI-assisted redesign focused on planting, shade, flexible seating, pedestrian comfort, and community use.

Project Goal

Harlem Plaza already functions as an important public area, but it can do more. This redesign asks how landscape design, public amenities, and visual identity can improve daily life.
